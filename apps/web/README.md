# @arcanum/web

Frontend приложение "Матрица Судьбы"

## Технологии

- React 18.2+
- TypeScript 5.x
- Vite 5.x
- React Router 6
- Zustand (state management)

## Разработка

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Собрать production build
npm run build

# Предпросмотр production build
npm run preview

# Линтинг
npm run lint
```

## Структура

```
src/
├── components/     # React компоненты
├── pages/          # Страницы приложения
├── hooks/          # Custom hooks
├── store/          # Zustand stores
├── utils/          # Утилиты
├── types/          # TypeScript типы
└── styles/         # CSS файлы
```
